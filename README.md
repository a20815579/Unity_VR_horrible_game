# Unity VR恐怖解謎遊戲 - 光二毛球
## 簡介
以成大著名的鬼故事「光二毛球」作為故事原型，玩家在VR世界中可體會身歷其境的恐怖氣氛，一步步解謎並還原事件真相。
## 系統環境
- 版本：Unity 2019.4.15 LTS
- VR模組：XR interaction toolkit、XR plugin management
- VR設備：Oculus Rift S
## 遊戲模式與操作
採用類似「還願」的模式，透過場景互動與人物對話來推進劇情，並不時穿插恐怖詭譎的橋段
